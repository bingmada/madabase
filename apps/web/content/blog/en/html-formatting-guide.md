# HTML formatting guide

Formatted HTML is much easier to inspect than compressed markup.

## Why format HTML

Readable indentation helps you verify nesting, spot missing tags, and review generated templates.

## Common use cases

- Embedded widgets
- Email templates
- Copied snippets
- Generated markup

## Keep one warning in mind

Formatting improves readability, but you should still check complex templates manually.

## Final takeaway

An HTML formatter is a fast way to clean up messy markup before review.
