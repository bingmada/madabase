# 如何格式化 JSON

格式化 JSON 可以让原始数据更容易阅读和调试。

## 为什么格式化有帮助

美化后的 JSON 更容易查看嵌套对象、数组和字段名称。

## 一个简单流程

1. 粘贴原始 JSON。
2. 运行格式化。
3. 校验结果。
4. 复制美化后的输出。

## 常见使用场景

- 阅读 API 响应
- 清理复制来的 payload
- 检查配置文件
- 比较两个数据快照

## 最后总结

JSON 格式化工具是前后端工作流里最常用的浏览器工具之一。
