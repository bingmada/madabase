# HTML 格式化指南

格式化后的 HTML 比压缩后的标记更容易检查。

## 为什么要格式化 HTML

清晰的缩进有助于确认嵌套关系、发现缺失标签，以及审查生成模板。

## 常见使用场景

- 嵌入式小组件
- 邮件模板
- 复制来的代码片段
- 自动生成的标记

## 一个提醒

格式化可以提升可读性，但复杂模板仍建议人工复查。

## 最后总结

HTML 格式化工具是清理杂乱标记、开始审查前的快速步骤。
