# Base64 指南

Base64 是一种把二进制或普通文本转换成 ASCII 安全文本的编码格式。

## Base64 常出现在哪里

- Token
- Data URL
- 邮件内容
- 嵌入式资源

## 编码与解码

编码会把文本转成 Base64，解码则会还原为可读内容。

## 一个重要提醒

Base64 不是加密，它只是编码。

## 最后总结

当你处理 payload、文件或传输安全文本时，Base64 编码解码工具非常有用。
