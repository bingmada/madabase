# Regex 初学者指南

正则表达式可以帮助你搜索、匹配、校验和转换文本。

## 初学者常见使用场景

- 匹配邮箱
- 查找数字
- 校验格式
- 提取重复模式

## 为什么需要 Regex Tester

测试工具可以让你先在真实文本上验证模式，再放进代码里。

## 学习 regex 的建议

从简单模式开始，多测试，再逐步增强表达式。

## 最后总结

只要多结合真实文本练习，正则表达式会更容易掌握。
