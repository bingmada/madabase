# 什么是 JSON？

JSON 是 JavaScript Object Notation 的缩写，是一种轻量级的结构化数据文本格式。

## 为什么 JSON 很重要

开发者常用 JSON 处理 API 响应、配置文件、日志以及系统之间的数据交换。

## 常见 JSON 结构

- 键值对对象
- 有序数组
- 字符串、数字、布尔值和 null

## 什么时候需要 JSON 工具

格式化工具适合提升可读性，校验工具适合发现语法错误，JSON 转 TypeScript 工具适合生成类型。

## 最后总结

如果你经常处理接口或 Web 应用，理解 JSON 是基础能力。
