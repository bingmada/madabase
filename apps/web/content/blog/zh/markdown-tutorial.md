# Markdown 教程

Markdown 是一种轻量级写作格式，常用于文档、README、更新日志和文章。

## 基础语法

- `#` 表示标题
- `-` 表示列表
- 反引号表示代码
- 中括号加括号表示链接

## 为什么 Markdown 很实用

它本身就易读，也很容易转换成 HTML。

## 发布前先预览

实时 Markdown 预览工具能帮助你在发布前发现结构问题。

## 最后总结

Markdown 是开发者和技术写作者最实用的格式之一。
