[FAQ]
What does JSON diff do?
A JSON diff tool compares two JSON documents and highlights the differences. It shows what was added, removed, or changed between the two versions.

What differences are highlighted?
Typically: additions (items in new but not in original), deletions (items removed), and modifications (changed values).

Can it handle nested objects?
Yes. The diff algorithm compares nested structures and highlights differences at any level.

Is the comparison recursive?
Yes. All nested objects and arrays are compared recursively.

Does it show the actual values?
Yes. The tool shows the original value, the new value, and the location in the JSON structure.

Is my JSON data uploaded?
No. All comparison happens locally in your browser.

What is the use case for JSON diff?
Common uses include: comparing API responses over time, debugging configuration changes, reviewing data migrations, and understanding code changes.
