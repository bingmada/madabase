[FAQ]
What does JavaScript formatter do?
A JavaScript formatter (JS beautifier) adds proper indentation, spacing, and structure to JavaScript code, making it readable.

Can it format minified JavaScript?
Yes. Minified code often has all spacing removed; the formatter restores readable structure.

Does it understand ES6+ syntax?
Yes. The formatter supports modern JavaScript including arrow functions, destructuring, template literals, and async/await.

Can it format TypeScript?
No. TypeScript requires specialized formatters like Prettier with TypeScript support.

Is my code secure?
No. All formatting happens locally in your browser.
