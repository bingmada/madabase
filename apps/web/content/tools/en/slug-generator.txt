[FAQ]
What is a URL slug?
A URL slug is the part of a URL that identifies a specific page in a human-readable way. It is typically derived from the page title and uses hyphens to separate words.

How do I create a good slug?
A good slug is: lowercase, uses hyphens to separate words, contains only letters and numbers, is descriptive of the page content, and is not too long.

Why use hyphens instead of underscores?
Search engines treat hyphens as word separators but often treat underscores as part of words. Use hyphens for better SEO.

How long should a slug be?
Keep slugs under 50-60 characters for best compatibility. Include the most important keywords early in the slug.

Should I include dates in slugs?
For blogs, date prefixes like /2024/01/post-name can be useful but may complicate URL structure. Many prefer clean slugs without dates.

Does it handle special characters?
Yes. The tool removes or converts special characters while preserving the core words. Non-ASCII characters are converted to ASCII equivalents when possible.

Can I customize the output?
This tool generates slugs following best practices. For custom formats, you can modify the output manually.

Is my text uploaded?
No. All slug generation happens locally in your browser.
