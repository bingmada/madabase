[FAQ]
What is a hash?
A hash is a fixed-size output generated from input data using a mathematical function. The same input always produces the same hash.

What hash algorithms are supported?
Common algorithms include MD5, SHA-1, SHA-256, SHA-384, and SHA-512. Each has different characteristics regarding speed and security.

What is MD5 used for?
MD5 produces a 128-bit hash. It is fast but considered cryptographically broken and should not be used for security purposes.

What is SHA-256 used for?
SHA-256 produces a 256-bit hash and is widely used for security applications, blockchain, and data integrity verification.

Are hashes reversible?
No. Hashes are one-way functions. You cannot reverse a hash to recover the original data.

What is a collision?
A collision occurs when two different inputs produce the same hash. Good hash functions minimize collisions.

Can I use this for password hashing?
This tool generates raw hashes. For password storage, use specialized password hashing algorithms like bcrypt, scrypt, or Argon2 with proper configuration.

Is my text uploaded?
No. All hash generation happens locally in your browser.
