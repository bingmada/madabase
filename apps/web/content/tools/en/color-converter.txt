[FAQ]
What color formats are supported?
Supported formats include: HEX, RGB, RGBA, HSL, HSLA, and sometimes named colors.

What is HEX color format?
HEX colors use hexadecimal notation with a # prefix. Example: #3498db represents RGB(52, 152, 219).

What is RGB color format?
RGB colors specify red, green, and blue values from 0-255. Example: rgb(52, 152, 219).

What is HSL color format?
HSL specifies hue (0-360), saturation (0-100%), and lightness (0-100%). Example: hsl(204, 70%, 53%).

How do I add transparency?
Use RGBA or HSLA formats. Add an alpha value from 0 (transparent) to 1 (opaque).

What is the alpha channel?
The alpha channel controls transparency. 0 = fully transparent, 1 = fully opaque.

Is my color data uploaded?
No. All conversion happens locally in your browser.
