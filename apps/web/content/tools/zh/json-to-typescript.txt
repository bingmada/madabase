通过示例 JSON 生成类型定义，可以更快完成 API 类型、DTO 和页面状态建模。
[HOW_TO_USE]
粘贴示例 JSON
使用有代表性的对象或接口返回数据。
---
生成接口
工具会推断基础类型、嵌套对象和数组。
---
按业务调整
生成后可以继续修改命名和可选字段。

[FAQ]

可以。MVP 版本会为对象字段和数组生成嵌套接口。
