[FAQ]
什么是 YAML？
YAML（YAML Ain't Markup Language）是一种人类可读的数据序列化格式。它常用于 Docker、Kubernetes 和 Ansible 等 DevOps 工具中的配置文件。

YAML 格式化工具做什么？
YAML 格式化工具（YAML 美化器）获取混乱或缩进不正确的 YAML，并重新格式化以获得一致的缩进、适当的间距和干净的结构。

为什么 YAML 缩进很重要？
YAML 使用缩进来定义结构。不正确的缩进会改变 YAML 的含义，并可能导致配置错误。格式化工具确保一致、正确的缩进。

它如何处理嵌套结构？
格式化工具保留并正确缩进嵌套对象和数组，保持正确的 YAML 结构。

它会校验 YAML 吗？
是的。格式化工具检查 YAML 语法并报告任何带行号的错误，帮助你修复问题。

它捕获哪些常见的 YAML 错误？
常见错误包括不一致的缩进、错误的列表语法、缺失的冒号和不正确的字符串引号。

可以格式化多文档 YAML 吗？
是的。用 --- 分隔的多文档 YAML 文件会被正确处理。

我的 YAML 上传了吗？
没有。所有格式化和校验都在浏览器本地进行。
