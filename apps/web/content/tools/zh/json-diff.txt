[FAQ]
JSON diff 做什么？
JSON diff 工具比较两个 JSON 文档并高亮显示差异。它显示两个版本之间添加、删除或更改的内容。

高亮显示哪些差异？
通常包括：添加（原版中没有但新版本中有的项）、删除（移除的项）和修改（更改的值）。

可以处理嵌套对象吗？
是的。diff 算法比较嵌套结构并高亮显示任何级别的差异。

比较是递归的吗？
是的。所有嵌套对象和数组都会递归比较。

它显示实际值吗？
是的。工具显示原始值、新值和在 JSON 结构中的位置。

我的 JSON 数据上传了吗？
没有。所有比较都在浏览器本地进行。

JSON diff 的用例是什么？
常见用途包括：比较不同时间的 API 响应、调试配置变更、审查数据迁移以及理解代码变更。
