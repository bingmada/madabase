[FAQ]
支持哪些大小写格式？
常见格式包括：小写、大写、标题大小写、句子大小写、camelCase、PascalCase、snake_case 和 kebab-case。

什么是 camelCase？
camelCase 将单词组合，第一个单词小写，后续单词首字母大写。例如：helloWorldExample。常见于 JavaScript 变量命名。

什么是 snake_case？
snake_case 使用小写字母，单词之间用下划线分隔。例如：hello_world_example。常见于 Python 和 SQL。

什么是 kebab-case？
kebab-case 使用小写字母，单词之间用连字符分隔。例如：hello-world-example。常见于 URL 和 CSS。

什么是 Title Case？
标题大小写将每个主要单词的首字母大写。例如：Hello World Example。常见于标题和标题。

我的文本上传了吗？
没有。所有转换都在浏览器本地进行。
