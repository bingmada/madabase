[FAQ]
什么是 JSON 转义？
JSON 转义将特殊字符转换为其转义序列等效项。例如，" 变成 \"，换行变成 \n。

哪些字符需要转义？
需要转义的字符包括：引号（"）、反斜杠（\）、正斜杠（/）、退格符、换页符、换行符、回车符和制表符。

为什么要转义 JSON？
在将 JSON 字符串插入代码、日志或其他 JSON 文档时需要转义，以避免破坏结构。

转义和编码有什么区别？
转义处理 JSON 特定的特殊字符。编码（如 base64）将整个字符串转换为替代表示。

可以反转义 JSON 吗？
可以。该工具可以同时转义和反转义 JSON 字符串。

我的文本上传了吗？
没有。所有转义都在浏览器本地进行。
