[FAQ]
什么是 HTML 编码？
HTML 编码将特殊字符转换为 HTML 实体代码。例如，< 变成 &lt;，& 变成 &amp;。

为什么要编码 HTML？
编码防止浏览器将内容解释为 HTML。这对于安全显示用户输入和防止 XSS 攻击至关重要。

哪些字符被编码？
通常：<、>、&、" 和 '。一些编码器也处理其他特殊字符。

什么是 XSS？
XSS（跨站脚本攻击）发生在攻击者通过未编码的用户输入注入恶意脚本时。HTML 编码可以防止这种情况。

可以编码用于 JavaScript 而不是 HTML 吗？
不。这个工具用于 HTML 编码。对于 JavaScript，使用 JavaScript 字符串转义是不同的。

编码和清理一样吗？
不一样。编码将字符转换为安全等效项。清理完全移除或拒绝危险内容。

什么时候应该编码？
在 HTML 中显示用户生成内容时都要编码。这包括评论、消息、用户名和任何动态内容。

我的文本上传了吗？
没有。所有编码都在浏览器本地进行。
