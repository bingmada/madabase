[FAQ]
SQL 格式化工具做什么？
SQL 格式化工具（SQL 美化器）获取混乱或格式不好的 SQL 查询，并添加适当的缩进、换行和大写以使其可读。

为什么要格式化 SQL？
格式化的 SQL 更容易阅读、调试和维护。它有助于识别缺失的关键字、放置错误的条件和复杂的嵌套结构。

支持哪些 SQL 方言？
格式化工具处理标准 ANSI SQL 和常见变体。对于特定数据库方言（MySQL、PostgreSQL、Oracle），某些语法可能有所不同。

它如何处理长的 WHERE 子句？
长的 WHERE 子句通常跨多行拆分，AND/OR 对齐以提高可读性。

可以自定义关键字大小写吗？
是的。选择大写、小写或首字母大写用于 SQL 关键字，同时保留标识符的大小写。

它会验证 SQL 语法吗？
格式化工具专注于格式化而非验证。它可能会标记一些明显的语法问题，但不是完整的 SQL 验证器。

可以格式化子查询吗？
是的。格式化工具处理嵌套子查询和 CTE（公共表表达式），并带有适当的缩进。

我的 SQL 上传了吗？
没有。所有格式化都在浏览器本地进行。
