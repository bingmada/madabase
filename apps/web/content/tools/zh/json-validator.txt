[FAQ]
JSON 校验器做什么？
JSON 校验器检查你的 JSON 文本是否符合正确的 JSON 语法规则。它识别语法错误并提供有用的错误消息和行号。

什么算作无效 JSON？
常见问题包括缺少逗号、键未加引号、尾部逗号、单引号而非双引号，以及未闭合的方括号或花括号。

为什么需要校验 JSON？
JSON 用于 API、配置文件和数据存储。无效的 JSON 会在所有这些场景中导致失败。使用前校验可以防止应用程序出错。

JSON 和 JavaScript 对象有什么区别？
JSON 是一种遵循特定语法规则的文本格式。JavaScript 对象字面量是 JavaScript 代码的一部分，具有略微不同的语法（如单引号、尾部逗号和函数）。

如何修复常见的 JSON 错误？
检查属性之间是否缺少逗号，确保所有字符串使用双引号，删除尾部逗号，并验证所有方括号和花括号是否正确闭合。

可以处理大型 JSON 文件吗？
可以。校验器处理各种大小的 JSON。非常大的文件可能需要一点时间，但所有处理都在浏览器本地进行。

它适用于 JSONP 吗？
不。这个工具只校验 JSON 数据。对于 JSONP 校验，你需要先提取 JSON 部分。

我的 JSON 数据安全吗？
是的。所有校验都在浏览器本地进行。你的 JSON 永远不会发送到任何服务器。
