[FAQ]
CSS 格式化工具做什么？
CSS 格式化工具（CSS 美化器）向 CSS 代码添加适当的缩进、间距和结构，使其可读和可维护。

有哪些格式化选项可用？
常见选项包括缩进样式（空格或制表符）、属性排序和删除空行。

它如何处理 CSS3 和现代特性？
格式化工具支持所有现代 CSS 特性，包括 flexbox、grid、自定义属性和媒体查询。

可以格式化 SCSS 或 SASS 吗？
不。这个工具格式化标准 CSS 语法。对于 SCSS/SASS，使用专门的预处理器工具。

我的 CSS 代码安全吗？
没有。所有格式化都在浏览器本地进行。
