[FAQ]
什么是 XML 格式化工具？
XML 格式化工具（XML 美化器）向 XML 文档添加适当的缩进和结构，使它们可读。

XML 什么时候还在使用？
XML 在企业系统、SOAP API、RSS 订阅源、Microsoft Office 文档和配置文件中很常见。

它会校验 XML 吗？
格式化工具检查基本 XML 语法并可能报告明显错误。

XML 和 JSON 有什么区别？
XML 是带有开始和结束标签的标记语言。JSON 是带有键值对的数据格式。JSON 在现代 Web API 中更常见。

可以格式化带命名空间的 XML 吗？
是的。命名空间和前缀会被保留并适当格式化。

我的 XML 数据安全吗？
没有。所有格式化都在浏览器本地进行。
