[FAQ]
JavaScript 格式化工具做什么？
JavaScript 格式化工具（JS 美化器）向 JavaScript 代码添加适当的缩进、间距和结构，使其可读。

可以格式化压缩的 JavaScript 吗？
是的。压缩代码通常移除了所有间距；格式化工具会恢复可读结构。

它理解 ES6+ 语法吗？
是的。格式化工具支持现代 JavaScript，包括箭头函数、解构、模板字面量和 async/await。

可以格式化 TypeScript 吗？
不。TypeScript 需要专门的格式化工具，如带有 TypeScript 支持的 Prettier。

我的代码安全吗？
没有。所有格式化都在浏览器本地进行。
