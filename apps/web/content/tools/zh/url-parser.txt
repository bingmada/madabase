[FAQ]
URL 解析器做什么？
URL 解析器将 URL 分解为其组成部分：协议、主机名、端口、路径、查询字符串和哈希片段。

URL 的组成部分是什么？
URL 由以下部分组成：方案/协议（https://）、主机名（example.com）、端口（:8080）、路径（/path）、查询字符串（?key=value）和哈希片段（#section）。

路径和查询参数有什么区别？
路径标识服务器上的资源（/products/123）。查询字符串包含参数（?id=123&color=blue）。

什么是哈希片段？
哈希（#section）引用页面内的特定部分。它不会向服务器发送请求。

为什么要解析 URL？
解析有助于：构建动态链接、提取参数、验证 URL 和理解 URL 结构。

可以修改和重建 URL 吗？
这个工具专注于解析。对于构建 URL，使用你编程语言中的 URL 构造方法。

我的 URL 数据安全吗？
没有。所有解析都在浏览器本地进行。
