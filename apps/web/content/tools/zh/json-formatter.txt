[INTRO]
JSON 格式化工具可以把紧凑的 JSON 数据转成易读的缩进结构，方便调试接口返回、配置文件和日志。无论你是前端开发者、后端工程师，还是需要处理 API 数据的产品经理，这款免费的 JSON 格式化工具都能帮助你快速完成工作。所有操作都在浏览器本地完成，你的 JSON 数据不会被上传到任何服务器。
[HOW_TO_USE]
粘贴你的 JSON
将接口响应、配置文件、日志文件或任何来源的 JSON 数据粘贴到输入框中。
---
选择格式化或压缩模式
阅读和调试时选择「格式化」，让 JSON 以规范的缩进和换行展示。需要传输或存储时选择「压缩」，去除多余空格减小体积。
---
校验语法并复制结果
点击格式化或压缩按钮后，系统会自动校验 JSON 语法。如果有语法错误会高亮显示具体位置，方便你快速定位问题。
[EXAMPLES]
{"name":"Madabase","active":true,"tools":30}
===
{
  "name": "Madabase",
  "active": true,
  "tools": 30
}
[FAQ]
JSON 格式化工具做什么？
它可以在浏览器本地完成 JSON 格式化、压缩与校验。支持将紧凑 JSON 美化为易读格式，也支持将格式化后的 JSON 压缩为最小体积。
为什么要用 JSON 格式化工具？
JSON 是现代 Web 开发中最常用的数据格式，但紧凑的 JSON 很难阅读。格式化工具可以帮助开发者快速阅读接口返回、排查配置错误、分析日志数据。
这个工具可以校验 JSON 语法吗？
可以。工具会自动检测 JSON 语法错误，并在界面上高亮显示具体错误位置，帮助你快速修复。
可以压缩 JSON 吗？
可以。工具同时支持「格式化」和「压缩」两种模式，满足阅读调试和数据传输的不同需求。
我的数据会上传到服务器吗？
不会。所有处理都在浏览器本地完成，JSON 数据不会离开你的设备。
